# Страницы

| Файл | Продукт | Особенности |
|---|---|---|
| `frontend/index.html` | ОСАГО | Hero с полем госномера (тизер), калькулятор Финуслуг `#calc`, лента 9 страховых, бенто, памятка ДТП, FAQ, CTA |
| `frontend/kasko.html` | КАСКО | Без калькулятора — тёмная hero-панель объясняет, почему калькулятора КАСКО не существует; таблица ОСАГО vs КАСКО; форма расчёта `#lead` |
| `frontend/property.html` | Имущество | Квартира / дом / ипотека; риски; форма с select «Что страхуем» |
| `frontend/travel.html` | Путешествия | Калькулятор Финуслуг (data-id из `config.calc.travel`), слот `.ph` под фото в hero |
| `frontend/business.html` | Бизнес | Отдельный стиль: `data-theme="dark"` + класс `page-biz`, направления 01–06, процесс, B2B-форма `#brief` |
| `frontend/office.html` | Запись в офис | Яндекс.Карта (включается ключом в config.js), галерея-заглушки, запись `#booking` |

## Общие блоки
Шапка (все виды страхования), мобильное меню, футер и модалка `#leadModal`
дублируются на каждой странице. Источник — `index.html`; правки разносить
генератором `frontend/build_pages.py` (контент страниц задан в нём же).

## FAQ
Каждая страница задаёт `window.ECS_FAQ = [{q, a}, …]` в инлайн-скрипте,
аккордеон строит `ui.js`.
